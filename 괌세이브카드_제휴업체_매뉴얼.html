📋 고객 카드 발급 과정

1단계: 발급 페이지 접속

www.guamsavecard.com/register

2단계: 정보 입력

• 고객명 (한글/영문)
• 계약여행사 선택
• 4자리 비밀번호 설정

3단계: 카드 발급 완료

• QR 코드 자동 생성
• 카드 링크 제공
• 즉시 사용 가능

🛍️ 제휴업체 할인 처리 방법

1단계: 고객 카드 확인

고객이 QR 코드 또는 카드 링크를 제시

2단계: 제휴처명 입력

카드 페이지 하단에 업체명을 정확히 입력

3단계: 할인 처리 완료

"할인 사용 처리" 버튼 클릭 → 완료 메시지 확인

💡 팁: 하루에 여러 번 사용 가능하며, 모든 사용 내역이 자동 기록됩니다

📱 QR 코드 사용 예시

QR 코드 샘플
(실제 카드에서 생성)

사용 시나리오

1. 고객이 스마트폰으로 QR 코드 제시
2. 직원이 카드 페이지에서 업체명 입력
3. 할인 적용 후 결제 진행
4. 시스템에 자동으로 사용 기록 저장
